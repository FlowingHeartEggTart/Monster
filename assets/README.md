# 资源文件说明

本目录用于存放应用所需的静态资源文件。

## 必需文件

### 怪兽资源
- `monster.png` - 怪兽静态图片（默认显示）
- `monster-eating.mp4` - 怪兽吃蛋糕动画视频（点击怪兽时播放）

### 图标文件（开发阶段可暂时忽略）
- `icon.png` (1024x1024) - 应用图标
- `splash.png` (1242x2436) - 启动画面
- `adaptive-icon.png` (1024x1024) - Android 自适应图标
- `favicon.png` (48x48) - Web 图标

## 使用说明

1. 将 `怪兽吃蛋糕.mp4` 重命名为 `monster-eating.mp4`
2. 放入 `assets` 目录下
3. 点击养成页面的怪兽即可播放视频

## 注意

在开发阶段，图标文件的缺失不会影响应用运行，只会产生一些警告。
生产环境打包前需要添加这些文件。

视频文件必须存在，否则应用会报错。
