## 产品理念

> "不是控制你吃什么，而是理解你为什么吃"

帮助用户在情绪和进食之间建立一个「暂停键」——不是靠意志力硬撑，而是通过觉察、接纳和替代，温柔地打断自动化的进食反应。

## 核心功能

### 1. SOS（暂停键）
- **90秒陪伴式对话**：情绪冲动来临时，怪兽陪你度过90秒
- **去羞耻化设计**：不评判、不责怪，只陪伴
- **结算选择**：完成后可选择喂养怪兽或点亮灯塔

### 2. 养成（怪兽陪伴）
- **电子宠物界面**：你的专属情绪小怪兽
- **正念学堂**：学习情绪性进食知识，答题获得蛋糕
- **点亮灯塔**：送出微光，获得蛋糕奖励
- **蛋糕经济系统**：完成任务获得蛋糕，喂养你的怪兽

### 3. 灯塔（社区陪伴）
- **灯塔地图**：看到此刻有多少人和你一样在努力
- **微光墙**：匿名祝福滚动显示
- **送微光**：给陌生人送一句温暖的话

## 技术栈

- **框架**：Expo + React Native
- **路由**：Expo Router（文件路由）
- **状态管理**：Zustand
- **动画**：React Native Reanimated
- **本地存储**：AsyncStorage
- **类型安全**：TypeScript

## 设计特色

### OtterLife风格
- 柔和的浅紫渐变背景
- 治愈系配色（粉色、淡蓝、暖黄）
- 软萌的圆角设计
- 无压力的视觉体验

### 心理学原理
- **90秒法则**：情绪冲动会在90秒内自然减弱
- **去羞耻化**：不使用"失败"、"控制"等评判性词汇
- **替代性安抚**：用陪伴和理解替代进食
- **社会支持**：通过灯塔功能感受"我不是一个人"

## 怪兽类型

- 颜色：柔粉色

### ☁️ 默默（安静系）
- 性格：话少、安静陪着、偶尔说一句
- 颜色：淡蓝色

### 💜 丧丧（共情系）
- 性格：有点丧、但很懂你、不评判
- 颜色：淡紫色

## 核心设计原则

1. **零判断**：不评价用户行为对错
2. **低认知负担**：无需思考、无需复杂输入
3. **即时反馈**：每一步都有视觉/情绪回应
4. **可退出**：用户随时可以离开，不制造失败感
5. **去羞耻化**：用温柔的语言替代评判性词汇
