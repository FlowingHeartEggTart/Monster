# 🚀 PauseLight 启动指南

## 快速开始

### 1. 确认依赖已安装
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm start
```

### 3. 在浏览器中查看
按 `w` 键，或访问终端显示的 URL（通常是 `http://localhost:8081`）

## 🎯 首次体验流程

### 第一步：Onboarding（欢迎流程）
1. 看到欢迎页："在这里，你不需要假装没事"
2. 回答3个问题：
   - 你通常什么时候会想靠吃来缓解情绪？
   - 那个瞬间，你更希望有人...？
   - 你希望你的小伙伴是什么性格？
3. 给你的怪兽起名字
4. 看到匹配结果动画
5. 自动进入主页

### 第二步：探索三个Tab

#### 🌸 养成Tab（主页）
- 看到你的怪兽在呼吸（动画效果）
- 怪兽会说话（随机台词）
- 两个功能卡片：
  - **正念学堂**：点击进入，阅读知识卡片，答题获得蛋糕🧁
  - **点亮灯塔**：点击跳转到灯塔页面

#### ⏸️ SOS Tab（暂停键）
- 看到怪兽在待机
- 点击"我需要缓一缓"
- 进入90秒对话（怪兽陪你）
- 对话结束后选择：
  - 把这份宵夜给小怪兽（喂养，获得蛋糕）
  - 去灯塔看看（点亮你的光）

#### 💫 灯塔Tab（社区）
- 看到灯塔在旋转
- 在线人数显示（模拟数据）
- 微光墙：匿名祝福滚动
- 点击"送一束微光"：
  - 可以留言（选填）
  - 送出后获得蛋糕🧁

### 第三步：体验核心功能

#### 完成一次SOS
1. 进入 SOS Tab
2. 点击"我需要缓一缓"
3. 等待90秒（或点击右上角"我想先退出"）
4. 选择"把这份宵夜给小怪兽"
5. 自动跳转到养成页，看到蛋糕+1

#### 完成正念学堂
1. 在养成Tab，点击"正念学堂"卡片
2. 阅读知识内容
3. 回答问题
4. 答对后看到鼓励语和蛋糕+1
5. 卡片变成"今日已完成"

#### 送出微光
1. 进入灯塔Tab
2. 点击"送一束微光给陌生人"
3. 输入留言（或留空使用随机祝福）
4. 点击"送出 ✨"
5. 看到你的消息出现在微光墙顶部
6. 获得蛋糕+1

## 🎨 设计亮点体验

### 动画效果
- **怪兽呼吸**：在养成页和SOS页，怪兽会缓慢呼吸
- **光晕动画**：怪兽和灯塔周围的光晕会脉动
- **光点闪烁**：灯塔页面有12个光点在闪烁
- **旋转动画**：灯塔在缓慢旋转

### 交互细节
- **顶部状态栏**：显示怪兽名字和蛋糕数量
- **按钮反馈**：所有按钮都有点击效果
- **模态弹窗**：从底部滑入的流畅动画
- **完成标记**：完成的任务会显示绿色✓

### 文案细节
- **去羞耻化**：不使用"失败"、"控制"等词汇
- **温柔陪伴**：用"陪你"、"等待"、"温柔"
- **时间感知**：根据时间显示不同问候语

## 📱 功能说明

### 蛋糕经济系统
- SOS成功：+1 🧁
- 正念学堂：+1 🧁（每日一次）
- 送微光：+1 🧁（每日一次）

### 每日任务
- 每天0点自动重置
- 正念学堂和点亮灯塔可以再次完成
- SOS没有次数限制

### 数据持久化
- 怪兽信息（类型、名字）
- 蛋糕数量
- 每日任务完成状态
- SOS成功次数
- 累计天数
